

## 나인투원 사전과제

나인투원 사전과제 어플리케이션입니다.

## Function

1. Kakao Open API - '키워드로 장소검색'

2. 현재 기기의 위치로 검색어 입력 후 0.5초 뒤에 자동 검색

3. 검색 결과가 10개 이상일 경우, Paging을 통해 10개씩 결과 표출
